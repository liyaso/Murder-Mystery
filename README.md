# Murder Mystery
Group Members: Liya Sojan, Zaim Shakeel 

This project is an interactive, fictional murder mystery in which the user plays as a detective and solves the mystery. The website will have five pages, each unveiling new evidence for the user to find and piece together. The mystery will begin on the landing page, where the scene will be set, a summary of what the user is about to experience, and a navigation guide to the rest of the website. The second page will display the case board, which is a detailed timeline of events surrounding the crime. From there, the third page will introduce all the suspects and allow the user to understand each suspect's motives, alibis, and connections to the victim. All the evidence that has been collected will be shown on the fourth page. Once the user has understood all the evidence and believes they know who the murderer is, the user will be guided to the last page. On this last page, it will be revealed who the real murderer is, what the motive was, and their version of events.

## User Experience 

Home Page: When the user first lands on the page, they can read what the mystery is about, see a navigation guide, and click on ‘Enter the Case’ to begin 

Case Board Page: The user will be able to go through the entire timeline and click on each event to reveal more details 

Suspect Page: User will be shown all the suspects, and when the user clicks on a suspect card, the card will flip and reveal more details about the suspect 

Evidence page: Any evidence collected from the crime scene will be shown on this page, with details of what the evidence is and where it is from 

Reveal Page: Once the user believes they know who the murderer is, the identity will be revealed on this page.
